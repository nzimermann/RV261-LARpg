1. Introduction

Thank you for downloading Stylized Penguin (Free) by Hosh Studio.

This package includes a stylized low-poly penguin character designed for real-time engines such as Unity.

The character is fully rigged and includes a small set of animations to help you quickly test and integrate it into your project.

This is a lite version of a larger animation pack.


---------------------------------------

2. Package Contents

The package includes:

- 1 stylized penguin character
- Fully rigged model
- 5 animations
- Shared texture atlas
- Prefab
- Demo scene


---------------------------------------

3. Setup Guide

Follow these steps:

1. Import the package into Unity.

2. Open the Demo Scene:

Assets/Hosh/Stylized Penguin (Free)/Demonstration

3. Drag the penguin prefab into your scene.

4. The model and animations are ready to use.


---------------------------------------

4. Usage

The penguin prefab includes:

- Rigged character
- Animator component
- Basic animation controller

You can trigger animations through the Animator or assign your own controller.


---------------------------------------

5. Technical Notes

- Designed for real-time use
- Lightweight model
- Shared texture atlas
- Compatible with Built-in Render Pipeline and URP
